Listas

Lista ordenada

Utilizamos a tag <ol> dentro dessa tag colocamos uma tag chamada <li>, "type="1|A|a|I|i" 1 sequencia de numeros, A sequencia de letras maiusculas,a sequencia de numeros minusculas, I sequencia de numeros romanos maiusculas e i sequencia de numeros romanos minusculas
Exemplo:

  1. Lista
  2. ordenada

Lista desordenada

Utilizamos a tag <ul> para fazer uma lista desordenada, dentro dessa tag colocamos a tag <li>
Exemplo:

  • Lista
  • desordenada

Lista de definição

Para fazer uma lista de definição utilizamos a tag <dl>, e dentro dessa tag temos o termo que é a tag <dt> e a definição que é a tag <dd>
