As principais formatação de textos

Negrito / Destaque

Existem duas formas de colocar o texto em negrito
ultilizando a tag <b> (maneira não semantica)
ou também a tag <strong> (maneira semantica)

Italico / Ênfase

Existem duas formas de colocar o texto em italico
ultilizando a tag <i> (maneira não semantica)
ou também a tag <em> (maneira semantica)

Texto marcado

Utlizamos a tag <mark> para marcar o texto

Texto grande e pequeno

Para poder colocar o texto grande
ultilizamos a tag <big>(Essa tag está obsoleta),
para poder ter o texto pequeno utilizamos a tag <small>

Texto deletado

Para deixar uma linha no meio da frase utilizamos a tag <del>

Texto inserido

Para deixar uma linha debaixo da frase utilizamos a tag <del>

Texto sobrescrito

Para colocar tipo uma equação com elevação como 33+100,
utilizamos a tag <sup>

Texto subscrito

Para colocar tipo a formula da agua H2O
